Acesso Bloqueado (Firebase)

O seu banco de dados Firebase está recusando a conexão. Isso é normal em projetos novos.

Siga estes passos:

  1. Vá no Console do Firebase > Firestore Database.
  2. Clique na aba Rules (Regras).
  3. Copie o código abaixo e clique em Publish.
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
